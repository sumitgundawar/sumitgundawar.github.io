# Sumit Gundawar - Portfolio

Personal portfolio website. Built with React, TypeScript, Tailwind CSS, and Vite. Deployed to GitHub Pages at [sumitgundawar.com](https://sumitgundawar.com).

## Tech Stack

- React 18 + TypeScript
- Vite 7
- Tailwind CSS 4 + shadcn/ui
- GSAP + ScrollTrigger for scroll animations
- Lenis for smooth scrolling
- Framer Motion for page transitions
- Recharts for data visualisation
- Google Analytics for tracking

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sumitgundawar/sumitgundawar.github.io.git
cd sumitgundawar.github.io
```

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Deployment

Pushes to `main` trigger a GitHub Actions workflow that builds and deploys to GitHub Pages automatically.
